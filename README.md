# Plataforma de Cursos Online

Sistema web de cursos online com interface moderna e foco em conversao. Ideal para infoprodutores e educadores que querem monetizar seu conhecimento.

## Preview

[Ver Demo](https://plataforma-de-cursos-online-tau.vercel.app/)

## Tecnologias

- React
- TypeScript
- CSS
- JavaScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Catalogo de cursos com categorias
- Pagina de detalhes do curso
- Area do aluno
- Design responsivo e moderno
- Interface focada em conversao de vendas
